Configuration:
Username: COIN_NAME wallet address
Password: any
Server: SITE
Ports:
NON_SSL_PORT (low diff - CPU, GPU)
NICE_PORT (high diff - Nicehash, big farms)
SSL_PORT (SSL connection)

AMD Claymore Miner Setup
ZECMiner64.exe -zpool SITE:NON_SSL_PORT -zwal your_wallet.rig1 -zpsw any

SSL/TLS connection on SSL_PORT

ZECMiner64.exe -zpool ssl://SITE:SSL_PORT -zwal your_wallet.rig1 -zpsw any
add "-allpools 1" to config if you have error: pool does not support Zcash address at login
